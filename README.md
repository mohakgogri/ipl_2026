# We only belive in Jassi Bhai

This is a Streamlit application.

## How to run locally

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   streamlit run app.py
   ```

## How to deploy on Streamlit Community Cloud

1. Push this repository to a public GitHub repository. (e.g. create a repository on GitHub, then run `git remote add origin <URL>`, `git branch -M main`, and `git push -u origin main`).
2. Go to [share.streamlit.io](https://share.streamlit.io/).
3. Click **New app**.
4. Select your GitHub repository, branch, and specify `app.py` as the main file path.
5. Click **Deploy!**
